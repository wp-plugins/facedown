== Description ==

FaceDown allows to rotate your blog to 90º, 180º or 270º with HTML5 and CSS3.

== Installation ==

To install WP Monsters:

1. Install FaceDown automatically or by uploading the ZIP file or unzip plugin file and upload to /wp-contet/plugins/. 
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings page
